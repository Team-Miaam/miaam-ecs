Type System plays a crucial role in schema definition and cloning. In schema definition when the default value of a particular type is not given it is fetched from the type system. It ensures consistency and less undefined values. So the default value of a newly declared type cannot be undefined. By providing a cloning method, it facilitates deep copying custom values of newly created types.
