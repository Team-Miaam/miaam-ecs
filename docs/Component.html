(readonly) schema :Object

Schema of the component Do not manually access or set the schema directly, even if you know what you are doing!
This library is intended and designed to be used using only accessible methods. Because of restrictions of private static variable of JS, schema is exposed. Do not take the advantage of this always use setSchema and getSchema methods.
